[markdown](http://wowubuntu.com/markdown/basic.html) 语法总结
================

+ **标题**（markdown支持两种形式）
  + setext形式是用*底线*：`=`(最高阶标题)，`-`（第二阶标题）
  + atx形式是用`#`：1到6个`#`,对应标题1到6阶
+  **引用**： 用`>`
+ **修辞和强调**：用`*`或 `_`(在需要强调的字两端出现，一个是斜体，两个是加粗)
+ **列表** ：无序列表用`*` `+` `-` (星，加，减)
+ **链接** : (支持两种形式)
  + 行内形式： `[名称](链接)`
  + 参考形式： 给链接定义一个名字，可多处复用。比如：
    `I like [sina][1] than [baidu][2].`
    `[1]: http://www.sina.com`
    `[2]: http://www.baidu.com `
+ **图片(和链接语法很像)**
  + 行内形式：  `![名称](链接)`
  + 参考形式： `I like [sina][1] than [baidu][2].`
               `[1]: http://www.sina.com`
+ **代码**
 + 用反引号` 来标记代码区段
 + 只要每行都缩进 4 个空格或是一个 tab
